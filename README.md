# Mykyta Cherevatyi, DevIt assignment

## Requirements

* Node.js >= 18
* Docker

## Start services

To check functionality, run this command:

    docker-compose up

To terminate apps:

    docker-compose down

## Local development 

To start the server, you should run:

    cd server
    npm install
    npm run dev

To start the frontend, you should run:

    cd frontend
    npm install
    npm run start
